# cgp-Hackathon-BackEnd


## API Reference

## - Welcome message

```http
  GET localhost:8080/

```
#### Response
"Login & Register server"

## - Login

```http
  POST localhost:8080/auth/signin

```

#### Body (raw (JSON))
```javascript
{
        "username": "testUser",
        "password": "12345678"
}
```
